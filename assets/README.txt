Drop your pet art in this folder. The pet auto-detects, in this order:

    pet.gif   (animated GIF — frames play automatically)
    pet.png
    pet.jpg / pet.jpeg
    pet.webp

Recommended size: up to ~110x110 px (larger images are scaled down, aspect
preserved). PNG/GIF with a transparent background looks best, since the pet
window itself is transparent.

With no file here, the pet draws a built-in placeholder coin mascot so it is
always visible.
